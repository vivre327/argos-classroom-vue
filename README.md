## Publish

- 작업 브랜치 `pub`

```sh
git checkout pub
```

- 작업 링크

  {port}: 서버기동시 Port 번호

  http://localhost:{port}/publish/list

- 작업 리소스
  - 작업 폴더: src/views/publish/
  - 템플릿: src/views/publish/Template.vue
  - 리스트 화면 : src/views/publish/PublishList.vue
